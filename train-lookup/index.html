How to use
If this is a fresh session:
  1. Paste your previously-saved raw scan data into the Raw Scan Data box below and click Load Raw Data. This restores the service database from a previous scan.
  2. Then click Scan Missing Only to fill in any new stations that weren't scanned before.
  3. Click Build Journeys to generate the journey database.
  4. Before closing, click Save Raw Data, copy the textarea contents, and save somewhere safe.
If this is a first-time scan:
  1. Click Scan All Stations. This will cost ~2,000 API hits.
  2. After each station completes, raw data auto-saves to the textarea — copy it periodically as insurance.
  3. When done, click Build Journeys, then Save Raw Data.
To scan a specific station manually:
  1. Use the Manual Station Scan section below. Useful for filling gaps without re-running everything.
