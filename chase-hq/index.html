How to get your own API key ▼
An Anthropic API account is separate from Claude.ai. At Chase HQ usage levels it costs pennies per month.
1
Go to the Anthropic Console
Open console.anthropic.com and sign in or create an account.
2
Add a payment method
Go to Settings → Billing and add a card. Typical Chase HQ usage is under £1/month.
3
Create an API key
Go to Settings → API Keys and click Create Key. Name it “Chase HQ”.
4
Paste it here
Copy the key (starts with sk-ant-), paste into the field above, and click Save Key.
Keep your key private. Don’t share it or paste it anywhere other than this settings panel.
