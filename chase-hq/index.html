How to get your own API key ▼
An Anthropic API account is separate from Claude.ai. At Chase HQ usage levels it costs pennies per month.
1
Go to the Anthropic Console
Open console.anthropic.com in a new tab and sign up or sign in.
2
Add a payment method
Go to Settings → Billing and add a card. You won't be charged until you use the API — typical Chase HQ usage is under £1/month.
3
Create an API key
Go to Settings → API Keys and click Create Key. Give it a name like "Chase HQ".
4
Copy and paste it here
Copy the key (it starts with sk-ant-), paste it into the field above, and click Save Key.
⚠ Keep your key private. Don't share it or paste it anywhere other than this settings panel.
